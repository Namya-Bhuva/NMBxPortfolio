## 🛠️ Tech Stack
This portfolio is built with a focus on performance and modern design principles:
- **Frontend Framework**: React 19 + Vite
- **Styling**: Pure Modern CSS with custom variables and a glassmorphism aesthetic
- **Animations**: Custom CSS keyframes and IntersectionObserver for scroll reveals
- **Icons**: Inline SVG / FontAwesome

## 🌟 Key Features
- **Cinematic Design**: Deep space aesthetic with emerald/cyan gradients and dark themes.
- **Scroll Animations**: Smooth reveal animations as elements scroll into view.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop devices.
- **Performance First**: Built without heavy animation libraries (like GSAP or Framer Motion) to ensure maximum speed and minimal bundle size.

## ⚙️ Running Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Namya-Bhuva/NMBxPortfolio.git
   cd NMBxPortfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000` (or the port provided in your terminal) to view it in the browser.
